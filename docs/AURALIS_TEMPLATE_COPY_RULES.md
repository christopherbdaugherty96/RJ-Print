# Auralis Template Copy Rules

Status: active project rule

## Purpose

Document how Auralis Digital may be used as a reference while keeping RJ Print as a separate business and website.

## Allowed

- Use Auralis Digital as a layout reference.
- Reuse structural ideas.
- Reuse navigation concepts.
- Reuse page organization concepts.
- Reuse workflow lessons learned.

## Not allowed without explicit decision

- Overwriting Auralis Digital.
- Rebranding Auralis Digital into RJ Print.
- Mixing the two businesses into one website.
- Reusing customer-facing copy that is no longer accurate.
- Reusing branding assets that belong to a different business identity.

## RJ Print identity rule

RJ Print should be presented as its own business.

Auralis Digital may act as the builder, consultant, or website provider, but RJ Print should have its own:

- Business identity
- Product categories
- Contact information
- Policies
- Product catalog
- Launch checklist

## Migration rule

If a Shopify theme, page structure, or design system is reused, treat it as a template copy. Replace branding, content, navigation, products, and policies with RJ Print-specific information before launch.
