# RJ Print Website Plan

Status: planning baseline

## Website goal

Create a clear, customer-friendly website for RJ Print that explains what the business makes, separates product types, supports custom quote requests, and helps customers understand pickup, shipping, and production expectations.

## Primary customer actions

1. Shop ready-made prints.
2. Browse made-to-order products.
3. Request a custom print quote.
4. Contact RJ Print for business or bulk orders.

## Recommended sitemap

- Home
- Shop
- Ready-Made Prints
- Made-to-Order Prints
- Custom Print Request
- Business / Bulk Orders
- Gallery
- About RJ Print
- FAQ
- Shipping / Pickup Info
- Contact

## Homepage structure

### Hero

Headline options:

- Custom & Ready-Made 3D Prints
- 3D Printed Products, Gifts, Decor & Custom Ideas
- Bring Your Ideas to Life with RJ Print

Subheadline:

RJ Print creates ready-made, made-to-order, and custom 3D printed products for gifts, decor, practical use, and personal ideas.

Primary buttons:

- Shop Prints
- Request a Custom Print

### Product lanes

Three main cards:

1. Ready-Made Prints
   - Already printed and available now.

2. Made-to-Order Prints
   - Pick a known product and request color, size, or finish options where available.

3. Custom Print Requests
   - Send an idea, photo, measurement, or problem and get a quote.

### Featured categories

Potential categories:

- Drinkware accessories / koozies
- Lamps and lighting
- Clocks
- Wallets and cases
- Home decor
- Gifts
- Desk accessories
- Custom parts and practical items
- Seasonal products
- Business/bulk prints

### Custom process section

1. Send your idea, photo, or measurements.
2. RJ Print reviews the request.
3. You receive a quote or next step.
4. Once approved, the item is printed.
5. Pickup or shipping is arranged.

### Trust section

- Locally made
- Custom options available
- Real product photos
- Practical and creative designs
- Quote review before custom work
- Pickup/shipping options

## Product page requirements

Each product page/card should include:

- Product name
- Price or starting price
- Category
- Photos
- Description
- Available colors/materials if known
- Approximate production time
- Pickup/shipping note
- Customization options
- Call to action

## Custom request page requirements

The custom request page should collect:

- Name
- Email
- Phone if needed
- Product idea
- Intended use
- Measurements or size
- Preferred color/material
- Deadline
- Pickup or shipping preference
- Budget range
- Photo/reference upload workflow if available

## FAQ topics

- How long does a print take?
- Can I request a custom design?
- Can you print from a photo?
- Do you ship or offer pickup?
- What materials do you use?
- Can I choose colors?
- Can you make replacement parts?
- Do custom requests require approval before payment?
- What happens if a print fails?

## Launch priority

First website version should focus on clarity and requests, not complex automation.

Minimum launch:

- Home
- Shop or Gallery
- Custom Print Request
- About
- FAQ
- Contact
- Shipping / Pickup Info

## Later features

- Shopify checkout
- Product filtering
- Online custom quote form with uploads
- Business/bulk order page
- Order status updates
- Printer queue integration
- Shopify-to-printer workflow

## Boundary

Do not build printer automation or custom checkout until the client confirms product categories, pricing, margins, and custom request rules.
