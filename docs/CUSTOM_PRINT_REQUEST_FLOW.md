# RJ Print Custom Print Request Flow

Status: planning baseline

## Purpose

Define how RJ Print should handle customer requests that are not simple ready-made or standard made-to-order products.

Custom requests must be reviewed before acceptance because print time, materials, fit, design work, file availability, and failure risk can vary heavily.

## Customer-facing process

1. Customer sends an idea, photo, measurement, file, or description.
2. RJ Print reviews the request.
3. RJ Print asks follow-up questions if needed.
4. RJ Print gives a quote or declines the request.
5. Customer approves scope, price, and timeline.
6. Payment/deposit is collected if required.
7. Item is printed.
8. Finished item is reviewed and prepared for pickup/shipping.
9. Customer receives item.

## Request form fields

Minimum fields:

- Name
- Email
- Phone optional
- What do you want printed?
- What will it be used for?
- Size or measurements
- Preferred color
- Preferred material if known
- Quantity
- Deadline
- Pickup or shipping preference
- Budget range
- Reference image/file link if available

## Internal review checklist

Before quoting, confirm:

- Is the request physically printable?
- Is a 3D model/file available?
- Does the item require modeling/design time?
- Are dimensions clear?
- Is the material appropriate?
- Is the expected finish realistic?
- Is there a safety, legal, or IP/copyright issue?
- How long will it take to print?
- What is the failure risk?
- Is assembly or finishing required?
- Does the client have capacity to complete it on time?

## Quote should include

- Product/request summary
- Quantity
- Estimated production time
- Price
- Deposit/payment requirement
- Pickup/shipping plan
- Revision or approval limits if design work is involved
- What is not included

## When to decline or delay

Decline or delay if:

- the request is unsafe
- the file is unusable
- the customer needs engineering guarantees RJ Print cannot provide
- dimensions are unclear
- material expectations are unrealistic
- copyright/logo rights are unclear
- timeline is too short
- price expectation is too low
- failure risk is too high for the budget

## Custom request boundaries

RJ Print should not guarantee replacement-part fit, load-bearing strength, food safety, heat resistance, child safety, or commercial compliance unless the client has verified material, design, and use-case requirements.

## Pricing structure

Custom quotes should include:

```text
base review/custom fee
+ modeling/design time
+ material
+ print time
+ labor/handling
+ failure allowance
+ finishing/assembly
+ packaging/shipping
+ rush fee if needed
```

## Customer copy draft

Have an idea for something custom? Send us a description, photo, measurement, or reference file and RJ Print will review the request. Custom prints are quoted before production so we can confirm size, material, timeline, and price.
