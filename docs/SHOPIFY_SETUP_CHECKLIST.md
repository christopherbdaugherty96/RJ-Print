# RJ Print Shopify Setup Checklist

Status: planning checklist

## Purpose

Track the steps required if RJ Print decides to use Shopify for product display, quote collection, or e-commerce.

## Store foundation

- [ ] Confirm Shopify will be used.
- [ ] Confirm domain decision.
- [ ] Confirm branding assets.
- [ ] Confirm contact information.
- [ ] Confirm whether launch is quote-first or checkout-enabled.

## Product setup

- [ ] Confirm product categories.
- [ ] Confirm product list.
- [ ] Confirm product photos.
- [ ] Confirm product descriptions.
- [ ] Confirm prices.
- [ ] Confirm SKUs if inventory is tracked.

## Collections

- [ ] Ready-Made Prints.
- [ ] Made-to-Order Prints.
- [ ] Custom Print Requests.
- [ ] Business / Bulk Orders.

## Checkout readiness

- [ ] Confirm payment processor.
- [ ] Confirm sales tax handling.
- [ ] Confirm shipping setup.
- [ ] Confirm pickup options.
- [ ] Confirm refund policy.
- [ ] Confirm custom-order payment rules.

## Website pages

- [ ] Home.
- [ ] Product gallery or shop.
- [ ] Custom request page.
- [ ] About page.
- [ ] FAQ.
- [ ] Contact page.
- [ ] Policy pages.

## Testing

- [ ] Mobile review.
- [ ] Desktop review.
- [ ] Contact form review.
- [ ] Custom request flow review.
- [ ] Test order if checkout is enabled.

## Boundary

Do not implement printer automation, Shopify-to-printer automation, custom backend checkout, or queue automation as part of the first website scope.
