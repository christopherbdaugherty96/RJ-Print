# RJ Print V1 Launch Scope Lock

Status: active scope lock / requires client approval before launch

## Purpose

Lock the first useful RJ Print website scope so the project does not drift into unsupported checkout, automation, or broad product complexity before real client data and approval exist.

## Default v1 launch mode

```text
Quote/contact-first by default.
```

Shopify checkout may be included only if RJ Print explicitly approves it and checkout readiness is proven with payment, tax, shipping or pickup, policy pages, product data, and a successful test order.

## Approved v1 goal

Launch a clear, professional RJ Print website that lets visitors:

- understand what RJ Print makes
- browse product categories or examples
- request a custom print quote
- understand pickup, shipping, and production expectations
- contact the business

## Approved v1 pages

- Home
- Product Gallery or Shop Preview
- Custom Print Request
- About RJ Print
- FAQ
- Shipping / Pickup Info
- Contact

## Approved v1 content sections

- Clear hero offer
- Product lanes: ready-made, made-to-order, custom requests
- Featured products or category examples
- Custom request process
- Gallery or product examples
- Trust and expectation section
- FAQ
- Contact/request call to action

## Allowed in v1

- Real product examples
- Real product photos
- Quote/contact form
- Clear custom request process
- Pickup/shipping expectations
- Product lanes and categories
- Basic Shopify setup only if verified
- Checkout only if explicitly approved and successfully tested

## Not allowed in v1

- Printer automation
- Shopify-to-printer automation
- Autonomous printer queue
- Automated quoting
- Custom backend checkout
- Unsupported public checkout
- File upload system unless a simple approved tool is selected
- Real-time order tracking
- Customer accounts
- Advanced inventory system
- Fake reviews or fake proof
- Unverified legal, tax, insurance, or automation claims
- High-risk product listings without review

## Product gate

No product should be promoted publicly unless it has:

- product name
- product lane
- category
- price or quote label
- material or material note
- production time or processing note
- pickup/shipping note
- photo or approved placeholder strategy
- risk review if needed

## Client approval gate

Before launch, RJ Print must approve:

- business positioning
- v1 launch mode
- page list
- product lanes and categories
- product prices or quote labels
- custom request process
- pickup/shipping expectations
- policy expectations
- final public copy
- final launch

If any planning doc conflicts with this file, treat this file and `docs/CURRENT_STATUS.md` as the current v1 launch boundary until intentionally updated.
