# RJ Print Current Status

Status: current truth layer

## Current truth

RJ Print is a client website and business-planning project for a 3D print shop.

This repo is currently a planning and business-launch binder for the client project. Shopify is the intended sales platform, but the Shopify store is not yet configured, populated, launched, or verified.

The repo does not yet contain a production website implementation, live storefront, working Shopify checkout, backend checkout, or printer automation.

## Placeholder rule

Anything marked `TBD`, `example`, `draft`, `to be confirmed`, `requires client confirmation`, `placeholder`, or `not finalized` must be replaced, approved, removed, or held before launch.

## Build boundary

Do not add printer automation, Shopify-to-printer automation, custom backend checkout, or file-upload complexity until the first client-approved website scope is stable.

Custom print requests should remain quote/review-first until manual request rules, pricing, and acceptance criteria are proven.

Keep Shopify checkout marked unverified until a test order is completed successfully.

## Definition of first useful version

A first useful version should let visitors:

- understand what RJ Print makes
- browse product categories or examples
- request a custom quote
- understand pickup/shipping expectations
- contact the business

## Definition of not ready

The project is not ready for public promotion until:

- contact path works
- client approves product prices and categories
- at least several product photos are available
- custom quote process is clear
- pickup/shipping/refund expectations are not vague
