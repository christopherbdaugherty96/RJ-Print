# RJ Print Launch Readiness Scorecard

Status: active launch-readiness tracker

## Purpose

Give RJ Print and the builder a simple way to see whether the project is ready for Shopify setup, checkout testing, and launch.

Use this as a quick dashboard. If an area is not confirmed, keep it at 0 or mark it as blocked.

## Scorecard

| Area | Score | Status | Notes |
|---|---:|---|---|
| Client handoff reviewed | 0/10 | Blocked | RJ Print must read the handoff README and service scope. |
| Product list | 0/10 | Blocked | Real launch products are not fully entered yet. |
| Product photos | 0/10 | Blocked | Product photos are required for Shopify product pages. |
| Product pricing | 0/10 | Blocked | Prices must be approved by RJ Print. |
| Cost and margin data | 0/10 | Blocked | Materials, print times, labor, packaging, and fees must be reviewed. |
| Shopify product data | 0/10 | Blocked | Titles, handles, collections, SKUs, variants, inventory, and SEO fields must be confirmed. |
| Shipping / pickup | 0/10 | Blocked | Rules must be confirmed before launch. |
| Policies | 0/10 | Blocked | Shipping, refund, custom order, privacy, and terms must be reviewed as needed. |
| Payment / tax setup | 0/10 | Blocked | Shopify payment and tax setup must be confirmed before checkout launch. |
| Checkout testing | 0/10 | Blocked | Checkout must remain unverified until a successful test order. |
| Custom request flow | 0/10 | Blocked | Custom work should remain quote/review-first until rules are approved. |
| Client approval | 0/10 | Blocked | RJ Print must approve launch scope, products, prices, policies, and Shopify settings. |

## Readiness levels

- 0-30: Planning only. Do not launch.
- 31-60: Discovery in progress. Product and policy details still missing.
- 61-80: Build preparation. Shopify setup may begin if product data is usable.
- 81-95: Pre-launch review. Test checkout, policies, and client approval must be complete.
- 96-100: Launch candidate, assuming no unresolved blockers remain.

## Launch rule

Do not launch or promote checkout until product data, photos, pricing, policies, shipping/pickup rules, Shopify payment/tax setup, checkout testing, and client approval are complete.
